**Project Title:**  StudyHub: A Mobile Learning Management System

**Project Description:** StudyHub is a mobile learning app designed to help students organize study materials, track academic progress, and maintain consistent study habits. It provides a centralized space for managing notes, viewing achievements, monitoring study hours, and tracking streaks. With its simple and user-friendly interface, the app supports effective and organized learning.

**Features:**


1. Features: User authentication (login & registration)


2. Study materials management (add, view, organize)


3. Favorite materials system


4. Learning progress tracking


5. Personalized dashboard (hours, streaks, completions)


6. Profile management (personal info & security)


7. Light/Dark mode toggle


8. Responsive UI and smooth navigation
